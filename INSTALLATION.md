# Installation

## Option 1: Install From VSIX (Recommended for Users)

1. Download `codex-notifier-private.vsix`.
2. Open VS Code.
3. Go to Extensions panel.
4. Click `...` (top-right) -> `Install from VSIX...`.
5. Select `codex-notifier-private.vsix`.
6. Reload VS Code when prompted.

## Option 2: Run in Extension Development Mode (For Contributors)

1. Open this project folder in VS Code.
2. Press `F5`.
3. In Extension Development Host, run:
   - `Codex Notifier: Test Sound`

## Quick Verify

1. Open Command Palette (`Ctrl+Shift+P`).
2. Run `Codex Notifier: Test Sound`.
3. Confirm sound + completion status UI behavior.
